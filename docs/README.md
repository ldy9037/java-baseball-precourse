## 기능 정의서
- 게임 흐름을 제어한다.
    - 게임을 시작한다.
    - 숫자를 입력한다.
    - 결과를 출력한다.
    - 게임을 종료한다.
- 게임을 생성한다.
    - 게임 상태를 확인한다.
    - 게임 결과를 확인한다.
- 숫자를 생성한다.
    - 숫자는 1부터 9사이의 값이다.
- 숫자 조합을 생성한다.
    - 각 숫자는 중복되면 안된다.
    - 숫자는 3자리만 허용된다.
- 결과를 생성한다.
    - 같은 수가 같은 자리에 있으면 스트라이크이다.
    - 같은 수가 다른 자리에 있으면 볼이다.
    - 같은 수가 없으면 아무것도 아니다.
- 힌트
    - 전체 결과를 확인한다.
